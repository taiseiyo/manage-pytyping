* *manage-pytyping*
** Overview
A tool for recording each pytyping score created by React.
 
If you want the score of your name to be output to the table, add the
name in the list of name_list in index.js.

** Start

#+begin_src bash
   git clone "URL"
   cd manage-pytyping
   npm install
   npm start
#+end_src
